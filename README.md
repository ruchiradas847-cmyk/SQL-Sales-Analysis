# SQL-Sales-Analysis
This project focuses on analyzing sales data using SQL.  Various queries were performed to extract meaningful insights such as total sales by category, top customers, and city-wise order distribution.  The project demonstrates the use of SQL concepts like SELECT, WHERE, GROUP BY, ORDER BY, and aggregate functions.

Tools Used - SQL


*Key Insights
- Identified top-performing product categories  
- Analyzed customer spending patterns  
- Evaluated order distribution across different cities
